# TEXT TO SPEECH CONVERTER
A simple web app that reads out your text in different voices and languages. 
Built this as a beginner JavaScript project to understand the Web Speech API.

What it does:
- Type anything in the text box
- Pick a voice or language from the dropdown
- Hit Listen and it reads it out loud
- Hit Stop anytime to cancel

Built with
- HTML
- CSS
- JavaScript (Web Speech API — no libraries, no frameworks)

How to run it
Just download the files and open index.html in Chrome.
Works best on Chrome — other browsers may have fewer voice options.
